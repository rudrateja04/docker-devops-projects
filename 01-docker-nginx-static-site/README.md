# Project 01: Docker Nginx Static Website

This project demonstrates how to build and run a static website using Docker and Nginx.  
A custom Docker image is created using a Dockerfile, and the application is served through an Nginx container with port mapping.

---

## Project Objective
- Understand Docker image and container concepts
- Learn how to write a basic Dockerfile
- Build a custom Docker image
- Run a container using port mapping
- Access the application via a web browser

---

## Tech Stack
- Docker
- Nginx
- HTML
