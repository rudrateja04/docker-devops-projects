# docker-devops-projects

This repository contains hands-on Docker projects that I built while learning and practicing Docker for DevOps.

Each project focuses on a specific Docker concept, starting from basic image creation to multi-container applications and production-ready Docker images.

---

## Projects

### 01. Dockerized Nginx Static Website
- Built a custom Docker image using Nginx
- Served a static website inside a Docker container
- Learned Dockerfile basics, image build, and port mapping

---

### 02. Docker Compose with Python and MySQL
- Built a multi-container application using Docker Compose
- Python Flask application connected to a MySQL database
- Learned service networking, environment variables, and container logs

---

### 03. Production-Ready Docker Image (Python)
- Built a production-ready Docker image using multi-stage Dockerfile
- Optimized image size using slim base images
- Used Gunicorn as the production server
- Tagged and pushed the image to Docker Hub

Docker Hub Image:
https://hub.docker.com/r/rudrateja04/python-prod-app
