This project demonstrates how to run a multi-container application using Docker Compose.  
A Python Flask application and a MySQL database run in separate containers and communicate with each other using Docker Compose networking.

---

## Project Objective
- Understand Docker Compose and why it is used
- Run multiple containers using a single configuration file
- Enable communication between application and database containers
- Use environment variables for database initialization
- Persist database data using Docker volumes
- Verify application access via browser

---

## Tech Stack
- Docker
- Docker Compose
- Python (Flask)
- MySQL
